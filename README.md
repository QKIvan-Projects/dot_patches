# Patches
## Use
First, please synchronize the source code of dotOS

```shell
cd sourcedirectory
```

Clone this repo
```shell
git clone https://github.com/coolscode/patches -b master dotos
```

Then apply the patches
```shell
bash anydirname/apply.sh dotos
```

There maybe more in the future...
